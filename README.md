# Cliptag

This is a simple app that uses CLIP to generate tags for images.

## On Hugging Face Spaces

https://huggingface.co/spaces/mtuckah/cliptag

## Up and running

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Gradio:

```bash
gradio gradpp.py
```
